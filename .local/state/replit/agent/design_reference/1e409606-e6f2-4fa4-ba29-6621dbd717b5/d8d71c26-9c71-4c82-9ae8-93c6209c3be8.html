Advanced React Patterns

1. Higher-Order Components (HOCs)

Higher-Order Components are a powerful pattern in React that allows you to reuse component logic. An HOC is a function that takes a component and returns a new component with enhanced functionality.

Code Example:

const withAuth = (WrappedComponent) => {
  return (props) => {
    const { isAuthenticated } = useAuth();
    
    if (!isAuthenticated) {
      return <LoginComponent />;
    }
    
    return <WrappedComponent {...props} />;
  };
};

2. Render Props Pattern

The render props pattern is a technique for sharing code between components using a prop whose value is a function. This pattern provides maximum flexibility and composability.

💡 Key Benefits:

  • Highly reusable and composable
  • Clear separation of concerns
  • Easy to test and maintain

3. Custom Hooks

Custom hooks are JavaScript functions that start with "use" and can call other hooks. They allow you to extract component logic into reusable functions.

useLocalStorage Hook Example:

const useLocalStorage = (key, initialValue) => {
  const [storedValue, setStoredValue] = useState(() => {
    try {
      const item = window.localStorage.getItem(key);
      return item ? JSON.parse(item) : initialValue;
    } catch (error) {
      return initialValue;
    }
  });

  const setValue = (value) => {
    try {
      setStoredValue(value);
      window.localStorage.setItem(key, JSON.stringify(value));
    } catch (error) {
      console.error('Error saving to localStorage:', error);
    }
  };

  return [storedValue, setValue];
};

4. Context API with Reducers

Combining React Context with useReducer provides a powerful state management solution for complex applications without the need for external libraries like Redux.

⚠️ Performance Considerations:

Be mindful of context value changes as they can cause unnecessary re-renders. Consider splitting contexts or memoizing values when needed.

Conclusion

These advanced React patterns provide powerful tools for building scalable and maintainable applications. Choose the right pattern based on your specific use case and requirements.
