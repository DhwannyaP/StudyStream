Chapter 3: Wave Functions and Probability

3.1 Introduction to Wave Functions

In quantum mechanics, the wave function (denoted as Ψ) is a mathematical description of the quantum state of a particle or system. The wave function contains all the information about the system that can be known, and its square magnitude gives the probability density of finding the particle at any given position.

Key Concept

The wave function Ψ(x,t) represents the amplitude of probability, not probability itself. The probability is given by |Ψ(x,t)|².

3.2 Normalization of Wave Functions

For a wave function to be physically meaningful, it must be normalized. This means that the total probability of finding the particle somewhere in space must equal 1:

∫-∞∞ |Ψ(x,t)|² dx = 1

This normalization condition ensures that the wave function represents a valid probability distribution across all possible positions where the particle might be found.

3.3 Time Evolution of Wave Functions

The time evolution of quantum systems is governed by the Schrödinger equation. For a time-independent Hamiltonian, the wave function evolves as:

Ψ(x,t) = Ψ(x,0) e-iEt/ℏ
